Zoomba jQuery SEO Optimized Dreamweaver Template

The Zoomba template is an extremely flexible Dreamweaver template. It takes the 960 Grid System and the jQuery javascript library and gives you an incredibly easy to customize template with jQuery interactivity.

Here's what's included

Here's what you get with the Zoomba template:

Dreamweaver templates

Separate template (.dwt) files are included for all four layouts:

  • Left sidebar
  • Right sidebar
  • Dual sidebars
  • No sidebars

PNG and PSD files

The vector source graphics are included, so whether you prefer Fireworks or Photoshop, you can alter the graphics to meet the needs of your project.

jQuery examples

All the content you see on these example pages is included in the template files. It gives you a jump start on creating your own masterpiece!

Non-template HTML files

Not everyone is a Dreamweaver user, so we also include the plain HTML files that can be edited with any HTML editor, not just Dreamweaver!

Zoomba is a SEO Optimized Template

Zoomba is extremely SEO-friendly, as the content code is always presented first to search engine spiders, rather than the sidebar code as in many templates. The code is W3C valid as well, so spiders will come back for more!

Based on the 960 Grid System

Zoomba is based on the 960 Grid System, which in a nutshell means it's EXTREMELY easy to customize your page layouts and columns. See how this page goes from a full-width column to four columns then back to two columns? It's very simple with the 960 Grid System!
